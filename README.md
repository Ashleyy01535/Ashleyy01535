# Hi there, I'm Ashley Yee

I am a senior pursuing a B.S. in Computer Science with a Specialization in Information Systems at the University of California, Irvine. I am a passionate full-stack developer with experience in cloud architecture, API design, and building robust, scalable web applications. 

## Experience

* **Software Engineer Intern** at **Badger Meter**: Modernizing backend infrastructure by refactoring legacy PHP into modular classes, implementing S3 migration scripts for site-wide storage on AWS, and optimizing system consolidation for internal dashboards.
* **Software Engineer Intern** at **SmartCover Systems**: Upgraded legacy PHP APIs to the modern Symfony framework with Swagger/OpenAPI documentation, managed Ubuntu server upgrades, and enhanced GIS spatial data visualization.
* **Software Engineer Intern** at **DOVR Media**: Developed and deployed production code to Google Cloud Functions to validate, process, and manipulate images.

## Featured Projects

* **Ethian Hounds (Large-scale ARPG User Board)**: A solo-developed, full-stack fantasy species platform featuring a complex account system, custom genetics inheritance roller, private messaging, and dynamic item trading.
* **MallowPets (Virtual Pet Adoption Board)**: A fictional pet adoption platform built with PHP and MySQL, featuring a robust custom forum system and dynamic pagination. 
* **Rott-Gutt (Professional Dynamic Art Portfolio)**: A MySQL-backed portfolio site with a secure admin dashboard enabling dynamic image sorting, theme management, and commission tracking.
* **Raven-Woods**: An indie game information portal equipped with an integrated account management system and internal messaging.
